# 简历快投助手

> 面向校招/网申场景的本地浏览器插件。它不做不可控的 OCR 自动填充，也不做正确率难保障的简历解析；它把所有字段、内容和顺序交给用户自己定义，作为招聘官网解析不完整时最高效的人工补充工具。

## 核心卖点

校招官网的简历解析经常会漏字段、错字段，尤其是项目经历、实习经历、自我评价这类长文本。与其依赖无法稳定控制正确率的 OCR 或自动解析，不如让用户自己维护一套最熟悉、最准确的字段库。

简历快投助手的重点就是：**所有分类、字段、内容、顺序都可以自定义**。用户可以按自己的简历结构保存常用信息，在网申页面需要补哪里，就点击哪里、一键粘贴哪里。

## 现有功能（当前代码可确认）

- 管理常用字段：支持分类、字段增删改、顺序调整，内容本地保存。
- 浮动面板：在当前页面右侧展示，可一键粘贴。
- 一键粘贴：用于常见文本输入框，减少重复复制粘贴。
- 体验优化：支持设置页与面板联动，支持 Tab 在面板内部快速切到下一条。

## 安装

- 打开 `chrome://extensions/`（或其他 Chromium 浏览器的扩展管理页）并开启开发者模式。
- 点击“加载已解压的扩展程序”，选择本项目根目录。
- 在工具栏点击插件图标即可打开浮动面板。

## 使用

1. 在“管理”里维护字段库。
2. 在目标招聘表单中先点击要填写的目标输入框。
3. 在浮动面板里选对应字段，点“粘贴”。

## 说明

- 本插件不提供 Word/JSON/PDF 简历解析。
- 不接入外部服务，数据默认保留在当前浏览器本地。
